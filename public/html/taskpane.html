Simulação de Monte Carlo

Instruções de uso

Para configurar uma célula como Entrada, clique no botão , após selecionar a célula.

Para configurar uma célula como Saída, clique no botão , após selecionar a célula.

Para configurar a distribuição de Entrada, selecione a célula e depois clique no botão .

Para configurar o número de iterações, clique no botão .

Para iniciar a simulação clique no botão .
